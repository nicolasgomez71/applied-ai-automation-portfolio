# Applied AI Automation Portfolio

Technical portfolio focused on practical projects in Python, SQL, APIs, SaaS support, workflow automation and applied generative AI.

## Objective

Build job-relevant technical skills for roles that can be performed from Colombia, without depending on US work authorization, visa sponsorship, relocation or physical presence in the United States.

## Target Roles

- Bilingual Technical Support Engineer in SaaS/AI
- AI Automation Specialist
- No-code / Low-code Automation Specialist
- SaaS Implementation Specialist
- Integrations Specialist
- QA Tester / Junior QA Automation
- Data / BI Analyst with applied AI
- Junior Python Developer with APIs and AI
- Junior RAG / LLM App Developer as a portfolio differentiator

## Core Skills

- Python
- SQL
- REST APIs
- JSON
- Postman / API testing
- Git and GitHub
- Technical troubleshooting
- SaaS support
- Documentation
- Workflow automation
- Generative AI APIs
- Basic RAG
- Embeddings
- Vector databases
- Power BI / reporting

## Portfolio Principle

This repository prioritizes employability over academic prestige.

The goal is to produce visible evidence of practical technical ability through small projects, documentation, troubleshooting exercises and applied AI workflows.
